# EthiopicWeb
Map English characters to Ethiopic phonetics <br />
```
Example 
```  
 HE | HU  | HI | HA | HEE | H | HO
--- | --- | ---|--- | --- | ---| ---
ሀ  | ሁ |   ሂ |   ሃ |  ሄ |  ህ | ሆ | 

#Usage

```javascript
1. <script src="src/ethiopicinput.min.js"></script>

2. <input id="myInput" style="width:300px;" />

3.   <script>
      $("#myInput").ethiopicInput();
     </script>
```
